// 2. 匹配与 Method 节点共享相同 fullPath 值的 Section 节点
MATCH (s:Section {fullPath: m.fullPath})
// 3. 建立关系：Method 节点属于该 Section
// 如果关系已存在，不会重复创建
MERGE (m)-[:BELONGS_TO_SECTION]->(s)
RETURN count(m) AS MethodsLinked
